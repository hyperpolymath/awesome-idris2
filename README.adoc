____
A curated, actively maintained list of awesome https://www.idris-lang.org/[Idris 2] resources, libraries, tools, and projects.
____

Idris 2 is a purely functional programming language with first-class dependent types, quantitative type theory, and multiple backends. It is the successor to Idris 1, rebuilt from the ground up with a self-hosting compiler written in Idris 2 itself.

*This list focuses on Idris 2.* Classic Idris 1 resources are preserved in a <<idris-1-legacy,legacy section>> for reference.

_Maintained by https://github.com/hyperpolymath[Jonathan D.A. Jewell]. Contributions welcome — see link:CONTRIBUTING.md[CONTRIBUTING.md]._

Wondering how this works? See link:EXPLAINME.adoc[].

== Books

* https://www.manning.com/books/type-driven-development-with-idris[Type Driven Development With Idris] — Edwin Brady. The definitive book on dependent types in practice (covers Idris 1 but concepts transfer directly).
* https://www.cs.kent.ac.uk/people/staff/sjt/TTFP/[Type Theory and Functional Programming] — Simon Thompson. Foundational type theory.
* http://www.cse.chalmers.se/research/group/logic/book/book.pdf[Programming in Martin-Löf's Type Theory] — Bengt Nordström et al. The theoretical underpinning.
* https://idris-hackers.github.io/software-foundations/pdf/sf-idris-2018.pdf[Software Foundations] — Formal proofs in Idris. https://github.com/idris-hackers/software-foundations[Repo].
* https://leanpub.com/gidti[Gentle Introduction to Dependent Types with Idris] — Accessible introduction for newcomers.
